FAQ

1.What is the Document Object Model (DOM)?

DOM (Document Object Model) is a programming interface for HTML documents. DOM allows programmatic access to the method tree. With them, you can change the structure, style or content of the document.

2.How do you select an element from the DOM?

getElementById(),getElementsByClassName(),querySelector(),querySelectorAll()

3.What is event delegation in the context of the DOM, and why is it useful?

Events in JavaScript are some of the user interfaces built into HTML. For example: Clicking a button, sending data to an input field and loading a webpage are also called an event. There are different types of events. For example: click, mouseover, mouseout,keyup etc

4.How do you manipulate an element's attributes and styles using the DOM?

We use these methods to get, set, or remove properties of an element: getAttribute() setAttribute() removeAttribute()
